## v0.1.0

### api-mocker 基于 `Tauri + Vue 3` 的本地 API Mock 与请求调试工具，面向前端联调、接口演示和本地自测场景。

#### Mock 服务器

- 本地 HTTP Mock 服务器，基于 Hyper 实现
- 支持自定义端口配置（默认 3000，范围 1024-65535）
- 服务器状态实时显示（运行中/已停止）
- 应用启动时自动启动 Mock 服务
- 支持路径匹配规则：
  - 精确匹配：`/api/users`
  - 通配符匹配：`/api/*`
  - 路径参数：`/api/users/:id`

#### 接口管理

- 分组管理：支持创建、编辑、删除分组
- 嵌套分组：支持多级分组结构
- 接口 CRUD：创建、编辑、删除接口
- 拖拽排序：支持分组和接口的拖拽排序
- 拖拽移动：支持将接口拖拽到其他分组
- 接口搜索：支持按名称、路径、方法搜索
- URL 唯一性校验：同一方法+路径组合不可重复

#### 请求调试

- HTTP 方法支持：GET、POST、PUT、DELETE、PATCH
- URL 输入：支持相对路径和完整 URL
- Params 配置：可视化添加 URL 查询参数
- Headers 配置：可视化添加请求头
- Body 类型支持：
  - None：无请求体
  - JSON：JSON 格式数据，支持格式化
  - Form：表单数据
  - Raw：原始文本
- JSON 实时校验：编辑时实时检测格式错误
- 自动保存：请求配置自动保存到接口

#### 响应查看

- 状态码显示：HTTP 响应状态码
- 响应时间：请求耗时统计（毫秒）
- 响应头查看：服务器返回的 Headers
- 响应体展示：
  - JSON 格式化显示
  - 图片预览（PNG、JPG、GIF、WebP 等）
  - 二进制文件处理（Base64 编码）
- 流式响应支持：大文件渐进式加载

#### Mock 规则

- 多规则管理：每个接口支持多个 Mock 规则
- 规则切换：一键切换激活的规则
- 规则参数配置：
  - 规则名称
  - HTTP 状态码（200、400、500 等）
  - 响应延迟（毫秒）
  - 响应体（JSON 格式）
- Monaco 编辑器：专业的代码编辑体验
- 编辑器展开：支持全屏编辑模式
- JSON 格式化：一键格式化响应体

#### Mock.js 支持

- 完整 Mock.js 语法支持
- 文本类占位符：@cname、@name、@ctitle、@csentence 等
- 数字类占位符：@integer、@float、@natural、@boolean
- 日期类占位符：@date、@time、@datetime、@now
- 网络类占位符：@email、@url、@ip、@domain
- 其他占位符：@id、@guid、@image、@color
- 数组生成：`"list|5-10": [{}]` 语法
- JSON 注释支持：支持 `//` 和 `/* */` 注释
- 宽松 JSON 解析：支持单引号、尾随逗号、无引号键名

#### 环境变量

- 多环境管理：dev、test、prod 预置环境
- 自定义环境：支持添加新环境
- 变量配置：键值对形式配置变量
- 变量启用/禁用：单独控制每个变量
- 变量引用：`{{变量名}}` 语法
- 自动替换：发送请求时自动替换变量
- baseUrl 支持：相对路径自动拼接 baseUrl

#### 历史记录

- 自动保存：每次请求自动记录
- 历史信息：方法、URL、时间、状态码
- 详情查看：完整的请求/响应详情
- 快速恢复：点击历史记录恢复请求配置

#### 数据导入导出

- JSON 格式导出：导出所有分组、接口、规则
- 文件选择器：系统原生文件对话框
- 数据导入：从 JSON 文件恢复数据
- 数据校验：导入时验证数据格式

#### 界面设计

- 三栏布局：图标栏 + 侧边栏 + 主工作区
- 侧边栏宽度可调：拖拽调整宽度（200-400px）
- 深色/浅色主题：支持主题切换
- 响应式设计：适配不同窗口尺寸
- Tab 切换：请求调试、Mock 规则、环境变量、历史记录

#### AI 助手

- AI 悬浮按钮：Mock 规则编辑时显示
- AI 对话框：与 AI 交互生成 Mock 数据
